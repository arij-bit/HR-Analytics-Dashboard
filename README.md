# HR Analytics Dashboard

## About the Project

I built this project to understand how HR data can be analyzed to identify employee attrition patterns and other workforce trends. The dashboard focuses on converting raw employee data into meaningful insights that can help HR teams make better decisions.

The project is based on the HR Analytics dataset.
---

## Tools Used

- Microsoft Power BI
- SQL
- Microsoft Excel
- Power Query
- DAX
- Git & GitHub

---

## Dataset

**Dataset File:** `HR_Analytics_Dataset.csv`

The dataset contains employee-related information such as:

- Employee Age
- Gender
- Department
- Job Role
- Education
- Monthly Income
- Attrition
- Overtime
- Job Satisfaction
- Work-Life Balance
- Years at Company
- Performance Rating

---

## Data Preparation

Before starting the analysis, the dataset was reviewed to ensure data quality.

The following validation steps were performed:

- Checked for missing values
- Verified duplicate records
- Confirmed correct data types
- Reviewed column consistency

The dataset was already well-structured, so no major preprocessing was required.

The validated dataset is available as:

`HR_Analytics_Cleaned.csv`

---

## Dashboard Features

The dashboard includes:

- Total Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Monthly Income
- Average Years at Company
- Department-wise Attrition Analysis
- Job Role Analysis
- Gender Distribution
- Education Analysis
- Overtime Analysis
- Interactive Filters and Slicers

---

## Key Insights

- Compared employee attrition across different departments.
- Identified job roles with higher attrition.
- Analyzed the relationship between overtime and employee attrition.
- Studied employee demographics and salary distribution.
- Created an interactive dashboard to support HR reporting.
